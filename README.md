To use this aliases:
- Put the `.bash_git_aliases` file in your home directory
- Load it from `~/.bashrc` using following snippet:
  ```
  if [ -f $HOME/.bash_git_aliases ]; then
    source $HOME/.bash_git_aliases
  fi
  ```
